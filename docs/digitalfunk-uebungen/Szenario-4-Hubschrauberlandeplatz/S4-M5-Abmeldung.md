**Kommunikationsebene:** GF ↔ Trupp(s) + GF → Leitstelle

## Format A — Dialogskript (Einsteiger)

### Phase 1: RTH abgeflogen — LZ räumen

| Wer | Spricht |
|-----|---------|
| **GF** | „Angriffstrupp von Florian Altenhammer 44/1, kommen" |
| **ATF** | „Florian Altenhammer 44/1 von Angriffstrupp, kommen" |
| **GF** | „Angriffstrupp, RTH abgeflogen. Markierungsleuchten einsammeln, LZ räumen, Ausrüstung verladen. Kommen" |
| **ATF** | „Verstanden. Leuchten einsammeln, LZ räumen, Ausrüstung verladen. Kommen" |
| **GF** | „Richtig. Ende" |

### Phase 2: Fertigmeldung + Abmeldung

| Wer | Spricht |
|-----|---------|
| **ATF** | „Florian Altenhammer 44/1 von Angriffstrupp, kommen" |
| **GF** | „Angriffstrupp von Florian Altenhammer 44/1, kommen" |
| **ATF** | „LZ geräumt, Ausrüstung verladen. Kommen" |
| **GF** | „Verstanden. Zum Fahrzeug. Ende" |
| **GF** | „Leitstelle Neustadt von Florian Altenhammer 44/1, kommen" |
| **Leitstelle** | „Florian Altenhammer 44/1 von Leitstelle Neustadt, kommen" |
| **GF** | „Florian Altenhammer 44/1, Einsatz beendet. RTH abgeflogen, HLZ geräumt. Wir rücken ein. Kommen" |
| **Leitstelle** | „Verstanden. Florian Altenhammer 44/1 einrücken. Ende" |
| **GF** | „Ende" |

---

## Format B — Rollenkarten (Mittelstufe)

### Rollenkarte: GF [PFLICHT]

1. Erteile Abbaubefehl nach RTH-Abflug: Leuchten einsammeln, LZ räumen.
2. Empfange Fertigmeldung.
3. Melde der Leitstelle: RTH abgeflogen, HLZ geräumt, einrücken.

---

### Rollenkarte: ATF [OPTIONAL]

1. Warte auf Abbaubefehl nach RTH-Abflug.
2. Quittiere: Leuchten, LZ räumen, Ausrüstung verladen.
3. Melde nach Abbau.

---

### Rollenkarte: Leitstelle [PFLICHT]

Empfange Abmeldung. Bestätige mit „Verstanden. Florian Altenhammer 44/1 einrücken. Ende".

---

## Format C — Kombination (Fortgeschritten)
Übungsleiter: Format A. Teilnehmer: je eine Rollenkarte.

---

## Format D — Freies Szenario (Erfahren)

**Situation:** RTH ist abgeflogen. Räumt den HLZ und meldet euch bei der Leitstelle ab.

Wichtig: Abbaubefehl erst nach sicherem RTH-Abflug erteilen!
