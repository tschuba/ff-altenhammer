**Funk-Sprecher min:** 2 — **max:** 4

## Format A — Dialogskript (Einsteiger)

### Phase 1: LZ-Fertigmeldung von Trupp

| Wer | Spricht |
|-----|---------|
| **ATF** | „Florian Altenhammer 44/1 von Angriffstrupp, kommen" |
| **GF** | „Angriffstrupp von Florian Altenhammer 44/1, kommen" |
| **ATF** | „LZ abgesteckt, fünf Leuchten stehen, lose Gegenstände entfernt. Kommen" |
| **GF** | „Verstanden. Alle in Deckung, Sicherheitsabstand 50 Meter. Ende" |

### Phase 2: LZ-Bereitschaftsmeldung an Leitstelle

| Wer | Spricht |
|-----|---------|
| **GF** | „Leitstelle Neustadt von Florian Altenhammer 44/1, kommen" |
| **Leitstelle** | „Florian Altenhammer 44/1 von Leitstelle Neustadt, kommen" |
| **GF** | „HLZ bereit. 30×30 Meter, fünf Leuchten, Windrichtung West, hindernisfrei. Personal in Deckung. Kommen" |
| **Leitstelle** | „Verstanden. RTH wird eingewiesen. ETA noch drei Minuten. Ende" |
| **GF** | „Verstanden. Ende" |

### Phase 3: RTH nähert sich (Leitstelle informiert)

| Wer | Spricht |
|-----|---------|
| **Leitstelle** | „Florian Altenhammer 44/1 von Leitstelle Neustadt, kommen" |
| **GF** | „Leitstelle Neustadt von Florian Altenhammer 44/1, kommen" |
| **Leitstelle** | „RTH im Anflug, zwei Minuten. Alle Lichter auf LZ, kein Blendlicht Richtung RTH. Kommen" |
| **GF** | „Verstanden. Alle Lichter auf LZ, kein Blendlicht. Ende" |

### Phase 4: Landung (GF bestätigt)

| Wer | Spricht |
|-----|---------|
| **GF** | „Leitstelle Neustadt von Florian Altenhammer 44/1, kommen" |
| **Leitstelle** | „Florian Altenhammer 44/1 von Leitstelle Neustadt, kommen" |
| **GF** | „RTH hat gelandet. Übergabe an Rettungsdienst. Kommen" |
| **Leitstelle** | „Verstanden. Ende" |
| **GF** | „Ende" |

## Format B — Rollenkarten (Mittelstufe)

### Rollenkarte: ATF [OPTIONAL]

Melde wenn LZ fertig: abgesteckt, Leuchten stehen, Gegenstände weg.

---

### Rollenkarte: GF [PFLICHT]

1. Empfange Fertigmeldung ATF. Alle in Deckung, 50m Sicherheitsabstand anweisen.
2. Melde LZ-Bereitschaft an Leitstelle: Größe, Leuchten, Windrichtung, Personal in Deckung.
3. Empfange RTH-ETA-Update und Einweisungshinweise.
4. Bestätige Landung und Übergabe an RD.

---

### Rollenkarte: Leitstelle [PFLICHT]

1. Empfange LZ-Bereitschaftsmeldung. Teile ETA mit.
2. Informiere GF wenn RTH im Anflug: Kein Blendlicht Richtung RTH.
3. Empfange Landungsbestätigung.

---

## Format C — Kombination (Fortgeschritten)
Übungsleiter: Format A. Teilnehmer: je eine Rollenkarte.

---

## Format D — Freies Szenario (Erfahren)

**Situation:** LZ ist fertig. Meldet Bereitschaft, koordiniert mit Leitstelle, weist auf Sicherheitsregeln hin.

Folgendes muss kommuniziert werden:
- Fertigmeldung LZ mit allen Angaben (Größe, Leuchten, Wind, Hindernisse)
- Sicherheitsabstand anweisen
- RTH-Anflug begleiten (keine Blendlichter!)
- Landungsbestätigung an Leitstelle

> Wichtiger Sicherheitsaspekt: Annäherung an RTH **nur** auf Zeichen des Piloten — niemals selbstständig!
