# S5a · M1 · Alarmierung

> [!WARNING]
> **NUR ÜBUNG — DMO-MODUS (Direktbetrieb)**
> Diese Übung findet ausschließlich im Digitalfunk-Direktbetrieb (DMO) statt.
> **Kein echter Einsatz! Keine automatische Alarmierung anderer Einheiten.**

**Kommunikationsebene:** Leitstelle Neustadt → GF
**Funk-Sprecher:** 2 — [PFLICHT] Leitstelle, [PFLICHT] GF
**Dauer:** ca. 3 Minuten

---

## Format A — Dialogskript (Einsteiger)

| Wer | Spricht |
|-----|---------|
| **Leitstelle** | „Florian Altenhammer 44/1 von Leitstelle Neustadt, kommen" |
| **GF** | „Leitstelle Neustadt von Florian Altenhammer 44/1, kommen" |
| **Leitstelle** | „Tierrettung, Katze auf Baum, Gartenstraße 3, Altenhammer. Tierbesitzerin vor Ort. Ausrücken! Kommen" |
| **GF** | „Verstanden. Tierrettung, Katze auf Baum, Gartenstraße 3 Altenhammer, Tierbesitzerin vor Ort. Wir rücken aus. Ende" |

---

## Format B — Rollenkarten (Mittelstufe)

### Rollenkarte: Leitstelle [PFLICHT]
Alarmiere: Tierrettung, Katze auf Baum, Gartenstraße 3 Altenhammer, Tierbesitzerin vor Ort.

### Rollenkarte: GF [PFLICHT]
Quittiere vollständig.

---

## Format C — Kombination (Fortgeschritten)
Übungsleiter: Format A. Teilnehmer: je eine Rollenkarte.

---

## Format D — Freies Szenario (Erfahren)
**Leitstelle:** Alarmiere zur Tierrettung.
**GF:** Quittiere korrekt.
