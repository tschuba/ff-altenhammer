**Kommunikationsebene:** GF ↔ ATF

**Rollen:** [PFLICHT] GF, [PFLICHT] ATF, [OPTIONAL] Leitstelle

## Format A — Dialogskript (Einsteiger)

### Variante 1: Rettung erfolgreich

| Wer | Spricht |
|-----|---------|
| **ATF** | „Florian Altenhammer 44/1 von Angriffstrupp, kommen" |
| **GF** | „Angriffstrupp von Florian Altenhammer 44/1, kommen" |
| **ATF** | „Tier gesichert und an Tierbesitzerin übergeben. Unverletzt. Kommen" |
| **GF** | „Verstanden. Steckleiter verladen und zurück zum Fahrzeug. Ende" |

### Variante 2: Tier klettert höher — Aufgabe

| Wer | Spricht |
|-----|---------|
| **ATF** | „Florian Altenhammer 44/1 von Angriffstrupp, kommen" |
| **GF** | „Angriffstrupp von Florian Altenhammer 44/1, kommen" |
| **ATF** | „Tier ist beim Annähern höher geklettert. Rettung nicht möglich ohne Gefährdung. Kommen" |
| **GF** | „Verstanden. Leiter sichern, Abstand halten. Tierbesitzerin informieren. Ende" |
| **GF** | „Leitstelle Neustadt von Florian Altenhammer 44/1, kommen" |
| **Leitstelle** | „Florian Altenhammer 44/1 von Leitstelle Neustadt, kommen" |
| **GF** | „Rettung nicht möglich, Tier zu hoch geklettert. Tierbesitzerin informiert. Einsatz wird eingestellt. Kommen" |
| **Leitstelle** | „Verstanden. Ende" |
| **GF** | „Ende" |

---

## Format B — Rollenkarten (Mittelstufe)

### Rollenkarte: ATF [PFLICHT]
Melde Ergebnis: Tier gerettet und übergeben — oder: Tier höher geklettert, Rettung nicht möglich.

### Rollenkarte: GF [PFLICHT]
Empfange Meldung ATF. Entscheide: Abschluss oder Einsatzabbruch.
Bei Abbruch: Leitstelle informieren mit Begründung.

### Rollenkarte: Leitstelle [OPTIONAL]
Nur bei Variante 2 (Abbruch). Empfange Abbruchmeldung, quittiere.

---

## Format C — Kombination (Fortgeschritten)
Übungsleiter: Format A (beide Varianten). Teilnehmer: je eine Rollenkarte.

---

## Format D — Freies Szenario (Erfahren)
**Situation A:** Rettung gelingt. Meldet Ergebnis und Übergabe.
**Situation B:** Tier klettert höher. Wie wird die Entscheidung zum Abbruch kommuniziert?

> Bewertet: Korrekte Meldung des Ergebnisses, bei Abbruch vollständige Begründung an Leitstelle.
