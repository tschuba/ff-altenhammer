## Format A — Dialogskript (Einsteiger)

| Wer | Spricht |
|-----|---------|
| **GF** | „Leitstelle Neustadt von Florian Altenhammer 44/1, kommen" |
| **Leitstelle** | „Florian Altenhammer 44/1 von Leitstelle Neustadt, kommen" |
| **GF** | „Florian Altenhammer 44/1 an der Einsatzstelle. Hund in offenem Kanalschacht, ca. zwei Meter tief. Tier unverletzt, erschöpft. Kein Gasgeruch. Tierbesitzer vor Ort. Wir bereiten Rettung vor. Kommen" |
| **Leitstelle** | „Verstanden. Ende" |
| **GF** | „Ende" |

---

## Format B — Rollenkarten (Mittelstufe)

### Rollenkarte: GF [PFLICHT]
Melde: Standort, Lagebeschreibung (Schacht, Tiefe), Zustand Tier (unverletzt, erschöpft), Gasprüfung (kein Gasgeruch), Tierbesitzer vor Ort.

### Rollenkarte: Leitstelle [PFLICHT]
Empfange Lagemeldung. Kurze Quittierung.

---

## Format C — Kombination (Fortgeschritten)
Übungsleiter: Format A. Teilnehmer: je eine Rollenkarte.

---

## Format D — Freies Szenario (Erfahren)
**GF:** Melde Lagebild vollständig. Gasprüfung ist Pflicht in der Lagemeldung!
