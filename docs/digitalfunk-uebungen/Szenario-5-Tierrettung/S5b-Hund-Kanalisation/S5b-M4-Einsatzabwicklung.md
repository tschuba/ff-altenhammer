## Format A — Dialogskript (Einsteiger)

### Variante 1: Rettung erfolgreich

| Wer | Spricht |
|-----|---------|
| **ATF** | „Florian Altenhammer 44/1 von Angriffstrupp, kommen" |
| **GF** | „Angriffstrupp von Florian Altenhammer 44/1, kommen" |
| **ATF** | „Tier gesichert und aus Schacht gehoben. Unverletzt. Übergabe an Tierbesitzer. Kommen" |
| **GF** | „Verstanden. Steckleiter raus, Schachtdeckel sichern. Ende" |

### Variante 2: Tier verletzt — Tierarzt anfordern

| Wer | Spricht |
|-----|---------|
| **ATF** | „Florian Altenhammer 44/1 von Angriffstrupp, kommen" |
| **GF** | „Angriffstrupp von Florian Altenhammer 44/1, kommen" |
| **ATF** | „Tier gerettet. Hinterlauf verletzt, kann nicht laufen. Tierarzt erforderlich. Kommen" |
| **GF** | „Verstanden. Tier sichern, Tierbesitzer informieren. Ende" |
| **GF** | „Leitstelle Neustadt von Florian Altenhammer 44/1, kommen" |
| **Leitstelle** | „Florian Altenhammer 44/1 von Leitstelle Neustadt, kommen" |
| **GF** | „Tier gerettet, aber verletzt. Tierarzt anfordern, Hinterlaufverletzung. Kommen" |
| **Leitstelle** | „Verstanden. Tierarzt wird verständigt. Ende" |
| **GF** | „Verstanden. Ende" |

---

## Format B — Rollenkarten (Mittelstufe)

### Rollenkarte: ATF [PFLICHT]
Melde: Tier gerettet, Zustand (unverletzt / verletzt + Art der Verletzung).
Erinnere an: Schachtdeckel sichern.

### Rollenkarte: GF [PFLICHT]
Empfange Meldung. Bei Verletzung: Tierarzt über Leitstelle anfordern.
Schachtdeckel sichern anweisen.

### Rollenkarte: Leitstelle [OPTIONAL]
Nur bei Variante 2. Empfange Anforderung Tierarzt, bestätige.

---

## Format C — Kombination (Fortgeschritten)
Übungsleiter: Format A (beide Varianten). Teilnehmer: je eine Rollenkarte.

---

## Format D — Freies Szenario (Erfahren)
**Situation A:** Tier unverletzt. Meldet Rettungserfolg, Übergabe, Schacht sichern.
**Situation B:** Tier verletzt. Wie wird Tierarzt korrekt angefordert?
> Bewertet: Schachtsicherung in der Meldung, bei Verletzung Tierarztanforderung über Leitstelle.
