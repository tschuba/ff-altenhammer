## Format A — Dialogskript (Einsteiger)

### Mindestbesetzung (GF + 1 TF)

| Wer | Spricht |
|-----|---------|
| **GF** | „Angriffstrupp von Florian Altenhammer 44/1, kommen" |
| **ATF** | „Florian Altenhammer 44/1 von Angriffstrupp, kommen" |
| **GF** | „Angriffstrupp, Steckleiter in Schacht, Rettungsleine vorbereiten. Eine Person steigt ab, eine sichert oben. Fangsack bereithalten. Kommen" |
| **ATF** | „Verstanden. Steckleiter Schacht, Rettungsleine, eine absteigen, eine sichern oben, Fangsack. Kommen" |
| **GF** | „Richtig. Ausführen. Ende" |

### Vollbesetzung (GF + 2 TF)

| Wer | Spricht |
|-----|---------|
| **GF** | „Angriffstrupp von Florian Altenhammer 44/1, kommen" |
| **ATF** | „Florian Altenhammer 44/1 von Angriffstrupp, kommen" |
| **GF** | „Angriffstrupp, Steckleiter in Schacht, eine Person steigt mit Rettungsleine ab. Kommen" |
| **ATF** | „Verstanden. Steckleiter, eine Person absteigen mit Leine. Kommen" |
| **GF** | „Richtig. Ende" |
| **GF** | „Wassertrupp von Florian Altenhammer 44/1, kommen" |
| **WTF** | „Florian Altenhammer 44/1 von Wassertrupp, kommen" |
| **GF** | „Wassertrupp, Sicherung an der Schachtöffnung. Rettungsleine halten. Fangsack bereithalten. Kommen" |
| **WTF** | „Verstanden. Sicherung oben, Leine halten, Fangsack. Kommen" |
| **GF** | „Richtig. Ende" |

## Format B — Rollenkarten (Mittelstufe)

### Rollenkarte: GF [PFLICHT]
ATF: Steckleiter in Schacht, eine Person absteigen, Rettungsleine, Fangsack.
WTF (opt.): Sicherung oben, Leine halten.

### Rollenkarte: ATF [PFLICHT]
Warte auf Ruf. Quittiere. Steckleiter + absteigen + Rettungsleine + Fangsack.

### Rollenkarte: WTF [OPTIONAL]
Warte auf Ruf. Sicherung oben, Leine halten.

---

## Format C — Kombination (Fortgeschritten)
Übungsleiter: Format A. Teilnehmer: je eine Rollenkarte.

---

## Format D — Freies Szenario (Erfahren)
**GF:** Erteile Befehle für sichere Rettung. Sicherung oben und unten muss kommuniziert sein.
> Bewertet: Sicherungsaspekt explizit genannt, Fangsack/Leine im Befehl.
