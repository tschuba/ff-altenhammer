# OPT: Tierrettung

**Ergänzt:** M3 (Befehlsgebung), M4 (Einsatzabwicklung), M5 (Abmeldung)
**Zusätzliche Rollen:** [PFLICHT] ATF, [OPTIONAL] WTF, [OPTIONAL] STF
**Kombinierbar mit:** OPT-Wasser (beliebige Variante)

---

## Taktische Grundregel

> **Eigenschutz > Brandbekämpfung ≥ Tierrettung**
> Außenangriff startet erst nach Evakuierungsbestätigung — nicht vorher.
> **NICHT hinter fliehende Tiere stellen — Verletzungsgefahr!**
> Tierbesitzer einbeziehen: Tiere folgen bekannter Stimme besser.

---

## M3

EL erteilt zusätzlich zum Wasserversorgungsauftrag auch den Tierrettungsauftrag.
GF gibt beide Aufträge an eigene Trupps weiter.

### Phase 0: EL erteilt Tierrettungsauftrag (ergänzt M3-Basis Phase 1)

| Wer | Spricht |
|-----|---------|
| **EL** | „Florian Altenhammer 44/1 von Florian Flossenbürg 41/1, kommen" |
| **GF** | „Florian Flossenbürg 41/1 von Florian Altenhammer 44/1, kommen" |
| **EL** | „Florian Altenhammer 44/1, zusätzlich zur Wasserversorgung übernehmt ihr die Tierrettung. Tiere noch im Stall, Türen zu. Kommen" |
| **GF** | „Verstanden. Tierrettung übernehmen, Tiere im Stall. Kommen" |
| **EL** | „Richtig. Ende" |
| **GF** | „Verstanden. Ende" |

### Format A — Dialogskript

#### Befehl ATF (Tierevakuierung)

| Wer | Spricht |
|-----|---------|
| **GF** | „Angriffstrupp von Florian Altenhammer 44/1, kommen" |
| **ATF** | „Florian Altenhammer 44/1 von Angriffstrupp, kommen" |
| **GF** | „Angriffstrupp, Stalltüren öffnen, Tierbesitzer zur Evakuierung einsetzen. Tiere auf Koppel treiben. NICHT hinter fliehende Tiere stellen. Kommen" |
| **ATF** | „Verstanden. Stalltüren, Tierbesitzer, Tiere Koppel, nicht hinter Tiere. Kommen" |
| **GF** | „Richtig. Ausführen. Ende" |

#### Befehl WTF (Außenangriff — wartet auf Evakuierung)

| Wer | Spricht |
|-----|---------|
| **GF** | „Wassertrupp von Florian Altenhammer 44/1, kommen" |
| **WTF** | „Florian Altenhammer 44/1 von Wassertrupp, kommen" |
| **GF** | „Wassertrupp, C-Rohr vorbereiten, Außenangriff Nordseite. Wasser marsch erst nach Evakuierungsbestätigung. Kommen" |
| **WTF** | „Verstanden. C-Rohr bereit, warte auf Evakuierungsbestätigung. Kommen" |
| **GF** | „Richtig. Ende" |

#### Optional: STF (Koppeltor + Tierarzt)

| Wer | Spricht |
|-----|---------|
| **GF** | „Schlauchtrupp von Florian Altenhammer 44/1, kommen" |
| **STF** | „Florian Altenhammer 44/1 von Schlauchtrupp, kommen" |
| **GF** | „Schlauchtrupp, Koppeltor öffnen und sichern. Tierarzt über Leitstelle anfordern. Kommen" |
| **STF** | „Verstanden. Koppeltor öffnen, Tierarzt über Leitstelle. Kommen" |
| **GF** | „Richtig. Ende" |

### Format B — Rollenkarten

#### Rollenkarte: GF [PFLICHT]
- **ATF:** Stalltüren öffnen, Tierbesitzer einsetzen, Tiere auf Koppel, NICHT hinter Tiere stellen
- **WTF (opt.):** C-Rohr bereit — Wasser marsch erst nach Evakuierungsbestätigung
- **STF (opt.):** Koppeltor öffnen + Tierarzt anfordern

#### Rollenkarte: ATF [PFLICHT]
Stalltüren öffnen + Evakuierung. **Sicherheitsregel: Nie hinter fliehende Tiere stellen!**
Melde wenn alle Tiere evakuiert sind.

#### Rollenkarte: WTF [OPTIONAL]
C-Rohr Außenangriff [Seite] vorbereiten. **Warten auf Evakuierungsbestätigung** — erst dann Wasser marsch.

#### Rollenkarte: STF [OPTIONAL]
Koppeltor öffnen und sichern. Tierarzt über Leitstelle anfordern.

## M4

### Format A — Dialogskript

#### Phase: Evakuierung abgeschlossen → Außenangriff freigeben

| Wer | Spricht |
|-----|---------|
| **ATF** | „Florian Altenhammer 44/1 von Angriffstrupp, kommen" |
| **GF** | „Angriffstrupp von Florian Altenhammer 44/1, kommen" |
| **ATF** | „Alle Tiere evakuiert und auf Koppel gesichert. Stalltüren offen. Kommen" |
| **GF** | „Verstanden. Außenangriff freigegeben. Ende" |
| **GF** | „Wassertrupp von Florian Altenhammer 44/1, kommen" |
| **WTF** | „Florian Altenhammer 44/1 von Wassertrupp, kommen" |
| **GF** | „Wassertrupp, Tiere evakuiert — Außenangriff Nordseite, Wasser marsch! Kommen" |
| **WTF** | „Verstanden. Wasser marsch! Ende" |

### Format B — Rollenkarten

#### Rollenkarte: ATF [PFLICHT]
Melde wenn alle Tiere evakuiert und gesichert sind.

#### Rollenkarte: GF [PFLICHT]
Warte auf ATF-Fertigmeldung. Dann: Außenangriff freigeben (WTF Wasser marsch).

#### Rollenkarte: WTF [OPTIONAL]
Warte auf „Wasser marsch" — erst wenn Evakuierung bestätigt!
