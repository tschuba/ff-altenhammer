# S3 · M2 · Meldung bei Einsatzleitung

> [!WARNING]
> **NUR ÜBUNG — DMO-MODUS (Direktbetrieb)**
> Diese Übung findet ausschließlich im Digitalfunk-Direktbetrieb (DMO) statt.
> **Kein echter Einsatz! Keine automatische Alarmierung anderer Einheiten.**

**Kommunikationsebene:** GF → EL (Florian Flossenbürg 41/1)
**Kanal:** Betriebskanal
**Funk-Sprecher:** 2 — [PFLICHT] GF, [PFLICHT] EL (Florian Flossenbürg 41/1)
**Hinweis:** In diesem Modul KEINE Kommunikation mit Leitstelle Neustadt!
**Dauer:** ca. 3 Minuten

---

## Format A — Dialogskript (Einsteiger)

> GF ruft nicht die Leitstelle, sondern Florian Flossenbürg 41/1 an!

| Wer | Spricht |
|-----|---------|
| **GF** | „Florian Flossenbürg 41/1 von Florian Altenhammer 44/1, kommen" |
| **EL** | „Florian Altenhammer 44/1 von Florian Flossenbürg 41/1, kommen" |
| **GF** | „Florian Altenhammer 44/1 einsatzbereit, ein TSF, fünf Mann. Kommen" |
| **EL** | „Verstanden. Willkommen. Wartet auf Einsatzbefehl. Ende" |
| **GF** | „Verstanden. Ende" |

---

## Format B — Rollenkarten (Mittelstufe)

### Rollenkarte: GF [PFLICHT]
Melde dich bei EL Florian Flossenbürg 41/1 — nicht bei der Leitstelle!
Melde: Rufzeichen (Florian Altenhammer 44/1), Fahrzeugtyp (TSF), Mannschaftsstärke (x Mann), einsatzbereit.

### Rollenkarte: EL — Florian Flossenbürg 41/1 [PFLICHT]
Eine Unterstützungseinheit meldet sich an. Empfange die Stärkemeldung.
Antworte: „Verstanden. Willkommen. Wartet auf Einsatzbefehl. Ende"

---

## Format C — Kombination (Fortgeschritten)
Übungsleiter: Format A. Teilnehmer: je eine Rollenkarte.

---

## Format D — Freies Szenario (Erfahren)
**GF:** Ihr seid in Flossenbürg angekommen. Meldet euch bei EL Florian Flossenbürg 41/1. Gebt eure Stärke durch.
**EL:** Du bist Einsatzleitung. Eine Unterstützungseinheit trifft ein. Empfange Meldung und weise auf Einsatzbefehl hin.

> Bewertet: Meldung bei EL (nicht Leitstelle!), vollständige Stärkemeldung.
