## Format A — Dialogskript (Einsteiger)

> Wortlaut genau so sprechen.
> **Merke:** An der Einsatzstelle meldet sich GF bei Florian Flossenbürg 41/1 — nicht bei der Leitstelle!

| Wer | Spricht |
|-----|---------|
| **Leitstelle** | „Florian Altenhammer 44/1 von Leitstelle Neustadt, kommen" |
| **GF** | „Leitstelle Neustadt von Florian Altenhammer 44/1, kommen" |
| **Leitstelle** | „Stallbrand in Flossenbürg. Unterstützung Wasserversorgung angefordert. Melden Sie sich an der Einsatzstelle bei Einsatzleitung Florian Flossenbürg 41/1. Ausrücken! Kommen" |
| **GF** | „Verstanden. Stallbrand Flossenbürg, Wasserversorgung, Meldung bei Florian Flossenbürg 41/1. Wir rücken aus. Ende" |
