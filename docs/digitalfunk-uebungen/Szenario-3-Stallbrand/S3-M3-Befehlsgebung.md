## Format A — Dialogskript (Einsteiger)

### Phase 1: EL erteilt GF den Auftrag (Betriebskanal)

| Wer | Spricht |
|-----|---------|
| **EL** | „Florian Altenhammer 44/1 von Florian Flossenbürg 41/1, kommen" |
| **GF** | „Florian Flossenbürg 41/1 von Florian Altenhammer 44/1, kommen" |
| **EL** | „Florian Altenhammer 44/1, ihr übernehmt die Wasserversorgung. Wasserquelle erkunden und aufbauen. Kommen" |
| **GF** | „Verstanden. Wasserversorgung aufbauen, Wasserquelle erkunden. Kommen" |
| **EL** | „Richtig. Meldung wenn bereit für Wasser marsch. Ende" |
| **GF** | „Verstanden. Ende" |

### Phase 2: GF gibt Befehl an eigenen Trupp weiter (Betriebskanal)

| Wer | Spricht |
|-----|---------|
| **GF** | „Angriffstrupp von Florian Altenhammer 44/1, kommen" |
| **ATF** | „Florian Altenhammer 44/1 von Angriffstrupp, kommen" |
| **GF** | „Angriffstrupp, Wasserquelle erkunden und Schlauchleitung aufbauen. Kommen" |
| **ATF** | „Verstanden. Wasserquelle erkunden, Schlauchleitung aufbauen. Kommen" |
| **GF** | „Richtig. Ausführen. Ende" |

## Format B — Rollenkarten (Mittelstufe)

### Rollenkarte: EL — Florian Flossenbürg 41/1 [PFLICHT]
Erteile GF Auftrag: Wasserversorgung aufbauen, Wasserquelle erkunden.
*(OPT-Wasser: Spezifische Quelle nennen — je nach Variante)*
*(OPT-Tierrettung: Auch Tierrettungsauftrag erteilen)*
Abschlusswort: „Meldung wenn bereit für Wasser marsch."

### Rollenkarte: GF [PFLICHT]
**Schritt 1:** Empfange Befehl von EL. Wiederholen und quittieren.
**Schritt 2:** Gib Befehl an ATF weiter. Warte auf Quittierung.
*(OPT-Wasser: Wasserquelle und Maschinist-Befehl weitergeben)*
*(OPT-Tierrettung: Tierrettungsbefehl an ATF/WTF weitergeben)*

### Rollenkarte: ATF [PFLICHT]
Warte auf Ruf des GF. Befehl empfangen (Wasserquelle erkunden + Schlauchleitung), quittieren, ausführen.

---

## Format C — Kombination (Fortgeschritten)
Übungsleiter: Format A. Jeder Teilnehmer bekommt nur seine Rollenkarte.

## Format D — Freies Szenario (Erfahren)
**EL:** Erteile Wasserversorgungsauftrag. Weise auf Bereitschaftsmeldung vor Wasser marsch hin.
**GF:** Empfange Befehl korrekt und gib ihn sofort vollständig an deinen Trupp weiter.
**ATF:** Quittiere korrekt.

> Bewertet: korrekte Befehlskette EL → GF → ATF, vollständige Befehlsinhalte, korrekte Quittierungen.
